How to get your Apps Script URL
1Open your Google Sheet → Extensions → Apps Script 2Delete any existing code and paste the contents of ManuCost_AppsScript.js (download below) 3Click Save (floppy disk), then Deploy → New Deployment 4Type: Web App · Execute as: Me · Who has access: Anyone 5Click Deploy → copy the Web App URL (starts with https://script.google.com/macros/s/…) 6Paste that URL in the field below and click Connect
Pre-configured with the default ManuCost Sheets backend. You can replace it with your own Apps Script URL at any time.
